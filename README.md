# personal-site

My personal site and portfolio: https://personal-site-zeta-ashen.vercel.app

Built with Next.js and Tailwind, deployed on Vercel. It holds my project write-ups (Launch Studio, a GPT trained from scratch, scaling laws and LoRA, demand forecasting, a network emulation) and short notes on building an AI product.

## Run locally

```bash
yarn
yarn dev
```

Content lives in `data/`: projects in `data/projectsData.ts`, posts in `data/blog/`. Based on the tailwind-nextjs-starter-blog template.
